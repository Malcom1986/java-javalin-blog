# java-javalin-blog

[Demo on Heroku](https://java-javalin-blog.herokuapp.com/)

### Requirements

  * JDK 16
  * Gradle 7
  * GNU Make

### Setup

```sh
make setup
```

### Run server

```sh
make start
# Open http://localhost:3000
```
